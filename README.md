## Installation

### Clone repo

``` bash
# clone the repo
$ git clone https://github.com/muruswam/iims-aap.git iims-app

# go into app's directory
$ cd iims-app

# install app's dependencies
$ npm install
```

## Usage

``` bash
# serve with hot reload at localhost:4200.
$ ng serve

# build for production with minification
$ ng build
```
